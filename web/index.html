Cloudflare Configuration

1. Ensure your domain's nameservers point to Cloudflare.

2. Go to your Cloudflare Dashboard -> My Profile -> API Tokens.

3. It is critical that you create a User API Token (Custom Token) here, NOT an "Account API Token" under "Manage Account", and NOT a Global API Key.

4. Grant the token the following permissions to support Tunnels, DNS, and SSH Access:

  • Account DNS Settings: Read, Write
  • DNS Firewall: Read, Write
  • DNS View: Read, Write
  • Access - Apps and Policies: Read, Revoke, Write
  • Access - Apps: Read, Revoke, Write
  • Access - Identity Providers: Read, Write
  • Access - SSH Auditing: Read, Write
  • Cloudflare Tunnel: Read, Write (Crucial for Tunnels)
  • Zone - DNS: Read, Write (Crucial for Routing)
